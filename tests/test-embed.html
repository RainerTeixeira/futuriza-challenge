Página de Teste - Magic Banner

URL atual:

Esta é uma página de teste para o Magic Banner Plugin. Se houver um banner cadastrado para esta URL, ele aparecerá no topo da página.

Como testar:

  1. Certifique-se de que o servidor Next.js está rodando (npm run dev)
  2. Crie um banner no painel admin (http://localhost:3000/admin)
  3. Use a URL desta página como URL do banner
  4. Recarregue esta página
  5. O banner deve aparecer no topo

Recursos do script:

  • ✅ Carregamento assíncrono (não bloqueia a página)
  • ✅ Botão de fechar
  • ✅ Lembra que foi fechado (localStorage)
  • ✅ Ajusta padding do body automaticamente
  • ✅ Escape de HTML para segurança
  • ✅ Compatível ES5

Atributos personalizados:

  • data-url - URL customizada para buscar banner
  • data-api - URL da API (padrão: http://localhost:3000)
  • data-target - Seletor CSS onde injetar (padrão: body)
Exemplo de uso:
<script src="http://localhost:3000/magic-banner.js"></script>
